## Regras do Servidor Gmod

### 1. PropKill / PropPush
Matar ou empurrar outros jogadores com props é proibido.
> **Penalidade:** Kick + Ban Temporário (1 dia)

### 2. Crash
Qualquer ação que cause queda ou lentidão no servidor é proibida.
> **Penalidade:** Ban Permanente Imediato

### 3. Exploit / Bug Abuse
Usar falhas ou bugs do jogo para obter vantagem é proibido.
> **Penalidade:** Ban Permanente

### 4. Racismo
Comentários racistas ou qualquer forma de discriminação racial são estritamente proibidos.
> **Penalidade:** Ban Permanente Imediato

### 5. Nazismo
Simbolismo nazista, apologia ao nazismo ou qualquer conteúdo relacionado é proibido.
> **Penalidade:** Ban Permanente Imediato

### 6. Crime / Preconceito
Qualquer tipo de crime virtual ou comportamento preconceituoso (sexismo, lgbtfobia, etc) é proibido.
> **Penalidade:** Ban Permanente Imediato

### 7. Diversão e Respeito
Divirta-se! Mas sempre respeite os outros jogadores. Trate todos como gostaria de ser tratado.

### 8. Spam de Props
Spammar props no servidor é proibido. Use props com responsabilidade.
> **Penalidade:** Kick + Ban Temporário

### 9. Conteúdo NSFW
Qualquer tipo de conteúdo NSFW (conteúdo sexual, explícito ou impróprio) é estritamente proibido.
> **Penalidade:** Ban Permanente Imediato

### 10. Abuso de Poder
Se qualquer SuperAdmin, Moderador ou Admin abusar de poder, abra um Ticket no Discord para denunciá-lo.
> **Penalidade:** Investigação e Punição
